# Bot Discord pour le Suivi des Prix des Cryptomonnaies

Ce bot Discord, développé en JavaScript, vise à fournir le prix actuel d'une cryptomonnaie dans la devise de votre choix. Il se met à jour toutes les secondes grâce à une API.

## Commandes Disponibles

- Utilisez `/price <cryptomonnaie> <monnaie>` pour obtenir le prix actuel d'une cryptomonnaie dans une devise spécifique.
- `/help` affiche la liste des commandes disponibles.

Exemple d'utilisation : `/price Bitcoin USD` vous donnera le prix actuel du Bitcoin en dollars américains.

## Configuration

Avant de l'utiliser, assurez-vous d'avoir configuré les autorisations et les clés API nécessaires pour accéder aux données des cryptomonnaies.
